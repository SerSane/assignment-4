# Portfolio Website - Feras Alharbi

A modern, responsive portfolio website showcasing my work as a Computer Science student passionate about AI and Cybersecurity. This project demonstrates my web development skills, built with vanilla HTML, CSS, and JavaScript, featuring dynamic content and sophisticated user interactions.

## Project Description

I developed this portfolio website from scratch using vanilla HTML, CSS, and JavaScript. The project features comprehensive functionality that meets all Assignment 3 requirements, including GitHub API integration and performance optimizations:

### Core Features
- **Responsive Design**: I implemented a fully responsive layout optimized for desktop, tablet, and mobile devices
- **Dark/Light Theme**: Built an intelligent theme toggle system with persistent user preferences stored in localStorage
- **Dynamic Content**: Developed time-based personalized greetings and an interactive project filtering system
- **GitHub API Integration**: Integrated GitHub REST API to dynamically fetch and display my latest repositories
- **Form Validation**: Created a comprehensive contact form with real-time validation, auto-save, and user feedback
- **Performance Optimizations**: Implemented lazy loading, script deferring, and DOM query caching for faster load times
- **Loading States**: Designed professional loading animations and comprehensive error handling throughout the application
- **Smooth Animations**: Added elegant transitions, hover effects, and scroll-triggered animations

### Sections
- **Hero**: Implemented dynamic time-based greeting system that changes based on time of day
- **About Me**: Personal introduction section with interactive elements
- **Projects**: Built a filterable project showcase with expandable details and tech stack tags
- **Latest GitHub Projects**: Integrated GitHub API to automatically fetch and display my 6 most recently updated repositories
- **Skills**: Organized technical skills by category with a modern card layout
- **Daily Inspiration**: Created a quote system using a local array that randomly selects inspirational quotes
- **Contact**: Developed a dual-section contact area with information display and a fully functional contact form
